## Начало


#### 1. Склонировать репозиторий
  ```sh
  git clone https://github.com/HromasDev/3msd_bot.git
  cd 3msd_bot
  ```
#### 2. Устанвить зависимости
   ```sh
   npm install
   ```
#### 3. Указать переменные окружения

- Создаем файл .env в папке проекта, с указанными даннными для бота

   ```js
    TOKEN=TOKEN
    CHAT_ID=CHAT_ID
   ```
#### 4. Запустить бота
   ```sh
   npm run start
   ```
